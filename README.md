## Usage for developers

```bash
# if you have not installed gulp on global environment yet
$ npm install gulp -g
```

```bash
$ npm install
$ gulp
```

Then, load this extension from **chrome://extensions**.

## Dependencies

* node 6.9.2
* npm 3.10.9
* jquery 3.1.1

Set the jQuery file to `/src/lib/` as `jquery.js`.
